Remote Mamba

从这台电脑安全映射并打开另一台 Mac 的 Mamba。

Checking

Remote Mac

远程 Control Center 已映射

首次使用

  1. 两台 Mac 登录同一个 Tailscale 私人网络。
  2. 远程 Mac 开启「系统设置 > 通用 > 共享 > 远程登录」。
  3. 先在 Terminal 使用 SSH Key 成功登录一次。这个页面不会储存密码。
  4. 远程 Mac 双击安装过 Mamba Watchdog 后，再回这里按连接。
这里只保存机器名称、用户名和端口，资料位于本机并被 Git 忽略。Mamba 不会开放到公网。
