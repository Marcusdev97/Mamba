6. Project Safety Rules · 不可以破坏的规则

这些约束比 UI 名称或 Notion 字段更重要

① Account = 真实号码；Binding = Device::号码`wa_01` 只是在本机使用的 instance 名，两台电脑可以重复。
② STOP 属于 Contact同一个号码在任何项目退订，所有项目都必须停止发送。
③ Timeout 不自动重发Provider 没确认时先查 WhatsApp，避免客户收到重复内容。
④ Resume 只补未完成 Part已完成 Part 和其他已处理客户永远不重发。
⑤ SQLite 每台 Mac 独立不要用 iCloud / Dropbox / Git 同步 `.sqlite` 文件。
⑥ Notion 最终退出热路径Notion 可以慢、可以重试，但不能阻塞 WhatsApp 收发。
