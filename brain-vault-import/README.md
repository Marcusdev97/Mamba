# 市场库 Import — Property213 KB (2026-06-02)

136 个 Kuala Lumpur/Selangor 项目,从 Property213 knowledge base 标准化转出。

## 怎么放进 Obsidian

把 `市场库` 整个 folder 用 Finder 拖进你的 vault(iCloud 的 `mamba-brain/`),
和「盘资料」「区域情报」并排。打开 `_INDEX.md` 就是全部项目的索引表。

## 这个库的定位(重要)

- **市场库 = 参考,AI 不读。** 系统只读「盘资料」folder。这 136 个是你查询、
  比价、竞品对比、找新盘用的。
- **要主打某个盘时**:copy 那一页进「盘资料」→ 补 playbook(卖点人话/客户常问/红线)
  → 核对数字后把 `verified: false` 拿掉 → AI 开始会讲这个盘。
- 每页 frontmatter 有 `qa_flag` / ⚠️ 数据核对 section 的,是 List 页和 Detail 页
  数据打架的项目——报价前一定先和 developer 确认。
- 全库 `verified: false`:这是自动转换的资料,没人工核对过,别直接报数字给客户。

## 格式

- frontmatter: 机器可读字段(developer/tenure/价格区间/完工期/uid...)
- 正文: 户型价格**原文**、sales package、maintenance 等,原文保留不改写
- `_INDEX.md`: 按州/区排序的总表,Obsidian 里可点进每一页

## 更新

下次拿到新版 Property213 xlsx,丢给 Claude 讲一句「更新市场库」即可重转。
